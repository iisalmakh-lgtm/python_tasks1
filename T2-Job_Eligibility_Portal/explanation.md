# Job Eligibility Portal

## Description
This program checks whether a candidate is eligible for the next interview stage based on the company's requirements.

## Requirements
- The candidate must know Python.
- The candidate must have at least 2 years of experience (or projects), or have a Computer Science degree or a Bootcamp certificate.

## Skills Used
- User Input
- Variables
- if / else
- Nested if
- Logical Operators (or)
- String Methods (`strip()` and `lower()`)
